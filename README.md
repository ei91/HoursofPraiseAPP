Hours of Praise App
Overview
The Hours of Praise app is a Christian event-based mobile application designed to enhance user engagement and provide a seamless experience for attendees of our annual worship concert. Built with SwiftUI, the app integrates several key features to deliver vital information about the event, facilitate donations, and offer multimedia content, making it a comprehensive platform for both current and prospective users.

Key Features
Video Embedding: Access a library of past event videos, allowing users to relive previous performances and inspiring moments.
Donation Management: Users can easily contribute to our mission through one-time or recurring donations, utilizing integrated QR codes for quick transactions.
Event Countdown: A dynamic countdown timer shows the remaining time until the next concert, building anticipation among attendees.
Event Details: Provides comprehensive information about the concert, including dates, featured artists, and venue details.
Barcode Scanning: Simplifies event access by allowing users to scan their tickets directly within the app, ensuring a smooth entry process.
Technology Stack
Front-End: SwiftUI
Back-End: PHP (for server-side logic)
Database: MySQL (for managing user data and donations)
APIs: SendGrid (for email notifications), third-party services for QR code generation
Project Structure
Model: Defines the data structures used in the app.
View: Contains all user interface components built using SwiftUI.
Controller: Manages the app’s logic and user interactions.
